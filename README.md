# Sentinel-2 Super Resolution for Urban Planning in Utrecht
This project aims to enhance and compare the spatial resolution of publicly available Sentinel-2 satellite imagery (10 m) with Dutch PDOK orthomosaics (7.5-25 cm) as ground truth in the Utrecht municipal area. Using AI-based image super-resolution methods, more specifically ESRGAN (GAN-based) and Swin2SR (Transformer-based). Using Sentinel-2 data to support urban planning, infrastructure mapping, disaster monitoring, and environmental analysis.
